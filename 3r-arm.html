Dynamic Simulation of a Planar 3R Robotic Arm under Gravity

Jan 2024 - Apr 2024

CAD Python MATLAB Computer-Aided Design

Overview

This project involved the computational modeling and dynamic simulation of a 3R robotic arm influenced solely by gravitational forces, using the Euler-Lagrange formulation. The objective was to understand the mechanical behavior of articulated systems without external control influences, simulating the arm's response to gravity.

Technical Implementation

  • Applied Euler-Lagrange formulation for dynamic modeling
  • Developed mathematical model for 3-DOF planar robotic arm
  • Simulated gravitational effects on arm configuration
  • Used advanced numerical integration techniques
  • Analyzed temporal evolution of position and orientation

Results

Through advanced numerical integration techniques, the project provided insights into the temporal evolution of the arm's position and orientation, showcasing its practical implications for robotic control system design and predictive modeling.

Skills Demonstrated

Computer-Aided Design (CAD), Python Programming, MATLAB, Research Skills
